glorytun

Small, simple and secure multipath UDP tunnel.

Originally developed as a side project. It was quickly integrated into the OverTheBox product to guarantee internet connectivity: link aggregation, failover with or without backup, traffic shapping and robust MTU discovery.

slashinit

Minimal PID 1 for initramfs.

You can allow yourself to do weird things like redirect all the logs to /dev/kmsg or updating your system via a kexec directly from your init process when you have your whole system under control in production.

slurm-https

Simple HTTPS API for slurm.

This project uses the worst parts of the Go language to provide a HTTPS+json API to the excellent slurm scheduler. I had written this for a PoC that was prematurely aborted so I decided to open source it.

vbridge

X11 Cloud desktop software.

The remote desktop software used at OVH, CEA-DAM and other equally strange places. A new video compression algorithm has been invented for the occasion. The keyboard mapping is generated on the fly as you type, so it is never necessary to configure it (unlike the alternatives). Sessions are encrypted with TLS and can be shared. The subject is vast and big improvements are still possible, contact me.

durry

The only GPU-less driver for X.org that works without modelines!

It’s a small project written quickly but it’s very useful with vbridge.
